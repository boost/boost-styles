## Installation

Add this line to your application's Gemfile:

```ruby
group :development do
  ...
  gem 'boost-styles', git: 'https://github.com/boost/boost-styles.git'
  ...
end
```

Run bundle:

    $ bundle

Run the generator:

    $ rails g boost_styles:install
